# SDLC — Getting Started

A companion repo for the **AI Context Engineering Workshop** (Peoria, IL — May 7–8, 2026), focused on the software development lifecycle side of building real ministry tools with AI.

## About

This repo is a lightweight starting point for putting context engineering into practice across a typical SDLC: planning, building, reviewing, and shipping. It's meant to pair with the workshop materials and give you a place to land while you experiment.

For the main event details, see the [Context Engineering Workshop repo](https://github.com/chriskehayias/ContextEngineering).

## Getting Started

```bash
git clone <this-repo-url>
cd SDLC
```

From here, open the project in your editor of choice and start exploring. If you're attending the workshop, make sure you've completed the [Prerequisites & Setup Guide](https://github.com/chriskehayias/ContextEngineering/blob/main/_requirements/README.md) first.
